1. Navigating Directories

  • pwd — Show current directory path.
  • cd [directory] — Move to a specific directory.
  • cd .. — Move one level up.
  • cd / — Go to the root directory.
  • cd ~ — Go to the home directory.
  • ls — List files and folders.
  • ls [path] — List contents of a specific path.

2. Listing Files

  • ls — Basic listing of files and folders.
  • ls -l — Long listing with details.
  • ls -a — Show hidden files.
  • ls -lh — Human-readable file sizes.
  • ls /path/to/folder — List a different directory.

3. Creating Files and Folders

  • touch filename — Create a new empty file.
  • mkdir foldername — Create a new directory.
  • mkdir -p path/foldername — Create nested directories.

4. Copying and Moving

  • cp source destination — Copy a file.
  • cp -r source destination — Copy a directory recursively.
  • mv source destination — Move or rename a file or folder.
  • mv oldname newname — Rename a file or folder.

5. Deleting Files and Folders

  • rm filename — Delete a file.
  • rm -r foldername — Delete a directory and its contents.
  • rm -f filename — Force delete a file without prompt.
  • rmdir foldername — Delete an empty directory.

6. Viewing File Contents

  • cat filename — Display file contents quickly.
  • less filename — View file one page at a time.
  • more filename — Scroll through file contents.
  • head filename — Show first 10 lines.
  • tail filename — Show last 10 lines.

7. Editing Files

  • nano filename — Easy text editor.
  • vim filename — Advanced text editor.
  • gedit filename — Graphical text editor.

8. Permissions

  • ls -l — View file permissions.
  • chmod 755 filename — Set read/write/execute permissions.
  • chmod u+x filename — Give execute permission to the user.
  • chown user:group filename — Change file ownership.

9. Installing Software

  • sudo apt update — Update package list.
  • sudo apt install packagename — Install a package (Debian/Ubuntu).
  • sudo yum install packagename — Install using YUM (CentOS).
  • sudo dnf install packagename — Install using DNF (Fedora).
  • sudo snap install packagename — Install via Snap store.

10. Getting Help

  • man command — Manual page for a command.
  • command --help — Quick help for a command.
  • info command — Detailed help information.
  • whatis command — Short description of a command.
